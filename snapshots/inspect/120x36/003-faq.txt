│    How We Compare            ││ A >                                                                                  │
│    Atomic Updates (A/B)      ││   LevitateOS is a daily driver Linux distribution competing directly with Arch       │
│    Variants                  ││   Linux. It borrows Arch's philosophy (user maintains their own packages, minimal    │
│                              ││   base that users build up) but substitutes hours of compilation with minutes of     │
│ ▸ Rec Tooling                ││   extraction from pre-built Rocky Linux packages.                                    │
│                              ││   Dimension        Arch Linux                   LevitateOS                           │
│                              ││   ────────────────────────────────────────────────────────────────────────────────── │
│                              ││   Target User      Power users seeking control  Same                                 │
│                              ││   Philosophy       Minimal base, user builds    Same                                 │
│                              ││   Package Manager  pacman + AUR                 recipe (Rhai scripts)                │
│                              ││   Binary Source    Compiled from source         Extracted from Rocky 10 RPMs         │
│                              ││   Build Time       Hours (compiling)            Minutes (extracting)                 │
│                              ││                                                                                      │
│                              ││ Q >                                                                                  │
│                              ││   Why not just use Arch?                                                             │
│                              ││ A >                                                                                  │
│                              ││   Arch requires compiling AUR packages. That means build dependencies, compiler      │
│                              ││   toolchains, and waiting. LevitateOS extracts enterprise-tested Rocky RPMs in       │
│                              ││   minutes. Same philosophy, less waiting.                                            │
│                              ││   If you want to compile everything yourself, use Arch. If you want the same control │
│                              ││   without the compilation overhead, use LevitateOS.                                  │
